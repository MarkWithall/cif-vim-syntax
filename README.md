CIF Syntax File For gVim
==============

A gvim syntax highlighting for CIF timetable files.

![](CIFVimSyntax.png)

For further information on the CIF file format, see [ATOC's spec](http://www.atoc.org/clientfiles/File/RSPS5004%20v27.pdf).
